# Quokka Careers™ 🦘

**Ridiculously Good Jobs. Suspiciously Good Vibes.**

Quokka Careers™ is a premium, modern recruitment portal designed for classroom roleplay, interviews, and professional development. It serves as a bridge for students to "apply" for fictional roles within a digital agency called **Quokka Firm**, specifically supporting the development of a Campus Dashboard platform.

## ✨ Features

- **Premium Design**: Built with a custom Vanilla CSS design system (no framework dependencies).
- **Glassmorphism UI**: High-end visual aesthetics with frosted glass effects and smooth gradients.
- **Responsive Layout**: Fluid design that works perfectly on desktop, tablet, and mobile.
- **SEO Optimized**: Semantic HTML5 structure with proper meta tags and heading hierarchy.
- **Interactive Experience**: Smooth scrolling and hover animations for a polished feel.

## 🚀 Deployment Guide

This project is a static site and is ready to be deployed to GitHub and Vercel.

### 1. GitHub Repository
To host your code on GitHub:
1. Create a new repository on [GitHub](https://github.com/new).
2. Initialize git in your local project folder (if not already done):
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Quokka Careers Portal"
   ```
3. Link to your remote repository and push:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git branch -M main
   git push -u origin main
   ```

### 2. Vercel Deployment
To deploy live to the web using Vercel:
1. Go to [Vercel](https://vercel.com/new).
2. Connect your GitHub account.
3. Import the `YOUR_REPO_NAME` repository.
4. Click **Deploy**. Vercel will automatically detect the `index.html` and host it as a static site.
5. Your site will be live at `https://your-project-name.vercel.app`.
